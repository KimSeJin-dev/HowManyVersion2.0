# HowMany_교내 체육관 APP

교내 체육관 APP입니다. (Version 2.0)

기능
1. 간편한 코로나 명부작성 > qr코드 스캔에서 생성으로 변경하여 관리자앱애서 관리
2. QR코드를 통한 간편한 출입시스템(생성으로 변경)
3. 실시간 인원 파악 및 일주일 단위 이용인원파악 > 시간별 이용인원파악
4. 운동 스탑워치 제공 ( Kotlin )
5. 회원관리자 페이지 제공

외부 라이브러리 : Android chart , Circle View , QR code 
웹 개발자 : 웹페이지 및 웹서버 제공 

내부 sql DB를 통해 개인정보 저장
